Títulos

Para marcar o título do texto é usado as tags h1 até h6, quanto menor o número maior o tamanho da letra
Ex:

Essa é a tag h1.

Essa é a tag h6.

Parágrafos

A tag "p" é usada para fazer um parágrafo no texto.

Listas

Para listar elementos, existem as listas ordenadas e não ordenadas no HTML.

Listas ordenadas, numeram os itens(li). Já as não ordenadas, apenas marcam.

  • Item 1 lista não ordenada
  • Item 2 lista não ordenada
  1. Item 1 lista ordenada
  2. Item 2 lista ordenada

Links

Para usar os links na sua página, é usada a tag "a", nela é possível linkar para outra página, ou até mesmo itens na sua própia página como foi feito na lista lá em cima!

Quebra de linha

Para deixar o texto mais organizado, existe a tag hr, que deixa um traço.

Outras tags

Tag del. A tag del é usada para identificar o texto que foi excluído de um documento, mas retido para mostrar o histórico de modificações feitas no documento.

Exemplo. Lorem ipsum dolor sit amet consectetur adipisicing elit. Mollitia a odit aliquam quidem dolorum! Ipsum esse quam nihil quisquam molestias.

Tag font. Ela é usada para mudar a formatação da fonte, podendo escolher tamanho, cor, estilo...

A tag abbr indica uma abreviatura ou sigla, como "WWW" ou "NATO". Marcando-se abreviaturas que você pode dar informações úteis para os navegadores, corretores ortográficos, sistemas de tradução e indexadores. Podemos deﬁnir abreviações com o elemento abbr.
